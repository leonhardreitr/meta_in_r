Change the name of the study “Rose” to “Bloom”

data <-
  data |> 
  mutate(Author = ifelse(row_number() == 5, "Bloom", Author))

Create a new variable TE_seTE_diff by subtracting seTE from TE. Save the results in data.

data <- 
  data |> 
  mutate(TE_seTE_diff = 
           TE - seTE,.before = 1)

Use a pipe to (1) filter all studies in subgroup “one” or “two”, (2) select the variable TE_seTE_diff, (3) take the mean of the variable, and then apply the exp function to it. Access the R documentation to find out what the exp function does.

data |> 
  filter(
    subgroup != "three"

  mutate(cool_new_variable =
